## 🗓 项目节奏

每个周期：一个人 独立在 **4周完整1个产品**

* Week 1：选题 +  调研 + 方案设计 输出 PRD 文档(含 MVP)
* Week 2：用 Vibe Coding 工具专注 MVP 开发 + 测试(功能测试/集成测试)
* Week 3：发布 Pre 版本，邀请测内用户试用拿到反馈，进一步优化
* Week 4：发布 Release 版本， 撰写 PR 文章宣发，做用户增长/收集用户反馈，闭环产品用户增长飞轮

你的节奏我们会保证可控，不卷，只专注于提升真实能力。
