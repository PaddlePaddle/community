## 📝 如何贡献

若您计划为 PaddleMaterials 贡献新模块或重大改进，请遵循以下流程：

1. **新建 RFC 目录**：在 `rfcs/PaddleMaterials/` 下创建以模块命名的文件夹（如 `new_model/`）
2. **编写 RFC 文档**：参考 [RFC 模板](https://github.com/PaddlePaddle/community/blob/master/rfcs/design_template.md) 编写设计文档
3. **提交 Pull Request**：将 RFC 文档提交至本 community 仓库
4. **社区评审**：等待 PaddlePaddle 团队及社区成员评审
5. **实现与合入**：RFC 通过后，提交代码，在 PaddleMaterials 主仓库中实现并合入
