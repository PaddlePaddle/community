## 性能对比

### 基准测试

```php
fib(30)        : 递归斐波那契
factorial(100k): 大数循环
sum_range(10M) : 整数热循环
mandelbrot(5k) : 浮点运算
```

### 结果

| 基准 | 无 JIT | tracing JIT | 加速比 |
|:-----|:------:|:-----------:|:------:|
| `fib(30)` 递归 | 0.7232s | **0.6073s** | **1.19x** |
| `factorial(100k)` 循环 | 0.0105s | **0.0082s** | **1.28x** |
| `sum_range(10M)` 热循环 | 1.0054s | **0.7303s** | **1.38x** |
| `mandelbrot(5k)` 浮点 | 0.1017s | **0.0662s** | **1.54x** |

### 分析

- **热循环密集**的基准提升最大（sum_range +38%, mandelbrot +54%）
- **浮点运算**受益于消除类型检查和 handler dispatch
- **递归**提升较小（+19%），因为函数调用开销占主导
- 整体加速比 16%-35%，与主流架构的 JIT 收益一致
